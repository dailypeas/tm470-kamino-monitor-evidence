# Kamino-related Solana monitoring prototype

This repository is a public mirror of the private working repository used for my TM470 project. It was created so that the technical evidence referenced in my TMA03 submission could be reviewed.

The project is a small Python monitoring prototype for directly Kamino Lending-related activity on Solana. It uses two explainable rules to assess rapid activity and repeated failed transactions, producing JSON and CSV output for manual review.

The current evaluation uses synthetic test data. The prototype is not intended to detect fraud or identify malicious wallets.

The repository includes the source code, test fixtures, automated tests, saved outputs and reproduction instructions.
